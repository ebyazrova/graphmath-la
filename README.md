# GraphMath — Linear Algebra

This repository contains visual and geometric interpretations of key concepts in linear algebra, including projections, subspaces and least squares.

## Least Squares (OLS)

- Geometric interpretation as projection onto the column space  
- Decomposition of y into y∥ (in col(X)) and y⊥ (orthogonal residual)  
- Representation of col(X) as a plane in ℝ³ (for three observations)
